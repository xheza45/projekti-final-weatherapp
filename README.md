# projekti-final-weatherapp

Ky është një app i thjeshtë për motin dhe hartën:

Tregon temperaturën dhe motin e qytetit që zgjidhni.

Ka një hartë ku shfaqet qyteti, dhe marker-i pulsion me temperaturën.

Sfondo ndryshon ngjyrë sipas temperaturës.

Mund të kërkoni qytete dhe të klikoni mbi sugjerimet.

Rifreskon të dhënat kur tërhiqet poshtë (pull-to-refresh).

# Si e përdorim

Shkruani qytetin që doni.

Zgjidhni qytetin nga lista ose shtypni Search.

Shikoni temperaturën dhe motin në ekranin kryesor.

Klikoni Map Screen për të parë hartën me marker pulsues.

# Si ta instaloni

Klononi repo-n:

git clone https://github.com/xheza45/projekti-final-weatherapp.git
cd Projekti Final WeatherApp


# Instaloni varësitë:

npm install


# Startoni app-in:

npm start
